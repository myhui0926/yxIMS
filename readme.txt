优选广告资源库存管理系统：
优选广告作为云南规模最大的广告公司，拥有数量极其庞大的各类广告资源，例如公交候车厅(上千块)，高速路大牌，市内大屏LED，市内天桥LED(70块)，楼宇小牌(若干)，收费站云围城，太平洋汽车，腾讯汽车，腾讯房产，网易新闻APP，一点资讯等等。
那么如何确定某类资源的库存成了一项极其庞大的工作，所以业务员在售卖广告资源的时候，往往无法准确确认某类资源的真实库存情况，可能客户需要的广告位置已经没有库存了，所以一套及时准确反应所有资源库存的网络系统成为了一项紧迫的工作。
本系统旨在及时，准确的确认所有广告资源的实时库存情况，方便公司管理，节省媒介部门的工作量。
项目包含功能：
1、广告资源管理（广告位添加、删除、修改）；
2、单据录入
3、实时反应广告资源库存情况
